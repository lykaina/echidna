65536 x 2-byte unsigned memory "mem"
128 x 4-byte unsigned loop nesting memory "rec" (used in ops: N, O, P, Q, - )
1 byte counter for loop nesting memory "reccount"
4 GiB max program file size
----
Memory Ranges:
Range 0000-00FF	core routine memory, notation used inside subroutines
Range 0100-EFFF	subroutine memory, each using a 256-byte range
Range F000-FFFF	common memory
----
Memory Notation:
$xxxx	indirect memory address xxxx
@xxxx	direct memory address xxxx
=xxxx	literal xxxx
